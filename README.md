# postcss-rb
postcss implementation for Rails applications.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'postcss-rb'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install postcss-rb
```
